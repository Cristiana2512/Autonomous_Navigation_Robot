# Autonomous_Navigation_Robot
This project features an Arduino-based robot designed for autonomous navigation using two primary modes: Line Following and Obstacle Avoidance.

**Path Tracking**: Uses IR sensors to follow a predefined track with real-time corrections.

**Obstacle Detection**: Integrated HC-SR04 Ultrasonic sensor to detect and bypass objects in its path.

**Brain**: Arduino Uno

**Programming**: Developed in C++ using the Arduino IDE.
